Simon Game

Test your memory skills with this classic Simon game! Watch the sequence of colors and sounds, then repeat it back. Each level adds one more step to the sequence. How many levels can you complete?

Level 0

How to Play

  • Click "Start Game" to begin
  • Watch the sequence of colors that light up
  • Repeat the sequence by clicking the same colors in the same order
  • Each successful round adds one more color to the sequence
  • If you make a mistake, the game ends and you can start over
  • Try to get the highest score possible!
